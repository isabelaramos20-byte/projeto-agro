# projeto-agro
Colheita de Inovação
Sobre o Projeto

O projeto Colheita de Inovação é um site educativo que apresenta como a tecnologia, a inovação e a sustentabilidade contribuem para o desenvolvimento do campo e para a produção de alimentos de forma mais eficiente e responsável.

O objetivo é informar e conscientizar os visitantes sobre a importância das novas tecnologias na agricultura e seus impactos positivos para a sociedade e o meio ambiente.
Tecnologias Utilizadas

    HTML5
    CSS3
    JavaScript

Estrutura do Projeto

    index.html — estrutura do site.
    style.css — estilos e layout.
    script.js — interatividade e funcionalidades.

Funcionalidades

    Layout responsivo.
    Menu de navegação.
    Animações suaves.
    Quiz educativo.
    Cards informativos.
    Contadores animados.
    Atualização automática do ano no rodapé.

Tema

Colheita de Inovação destaca a relação entre tecnologia, sustentabilidade e desenvolvimento no campo, mostrando soluções modernas que ajudam a construir um futuro mais sustentável.
Autor

Desenvolvido para fins educacionais.
